# Audio Recording Studio Website

## Project Overview
This project is a website for an audio recording studio that aims to showcase the studio's services, provide booking options, and display previous work.

## Features
- **User-friendly interface**: Easy navigation for potential clients.
- **Service listings**: Clear details about the services offered, including pricing.
- **Booking system**: An integrated system for clients to book studio time.
- **Portfolio**: A showcase of previous recordings and projects.

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js (for backend)
- Express.js
- MongoDB (for database)

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/gaguinhomanzicsgo-byte/gravadora-de-audio.git
   ```
2. Navigate into the project directory:
   ```bash
   cd gravadora-de-audio
   ```
3. Install the required packages:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```
